## Project Description
This project presents a detailed audit of VLC Media Player, an open-source multimedia player. It covers its origin, license, philosophy, Linux footprint, ecosystem, and comparison with proprietary software.

## Scripts Included

### 1. System Identity Report
Displays system information such as OS, kernel, user, uptime, and license.

### 2. FOSS Package Inspector
Checks whether VLC is installed and displays its details.

### 3. Disk and Permission Auditor
Shows permissions and disk usage of important system directories.

### 4. Log File Analyzer
Counts occurrences of a keyword (like "error") in a log file.

### 5. Open Source Manifesto Generator
Generates a personalized open-source philosophy statement.
